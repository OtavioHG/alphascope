# Monitoramento V2

- Prometheus: `http://localhost:9090`
- Grafana: `http://localhost:3000`
- Health API: `GET /healthz`
- Runtime local: `python -m alphascope.cli runtime-status`

Monitore:

- CPU e memória
- drawdown diário
- win rate
- open positions
- ordens falhadas
- API offline
- auditoria de mudanças
